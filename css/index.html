CSS

CSS(Cascading Style Sheets)는 HTML을 꾸며주는 디자인 언어 스타일시트입니다.


css 선택자

float으로 인한 영역 깨짐 방지법

float을 사용하면 height 값을 0으로 인식합니다.

  1. 깨지는 영역에 똑같이 float을 사용합니다.(X) 모든 영역에 float을 사용하게 되고 정확히 깨지는 영역을 알 수가 없습니다.
  2. float의 성질을 차단하는 clear:both를 사용합니다. (X) 깨지는 영역을 정확히 알 수가 없습니다.
  3. float을 사용한 부모 박스한테 overflow: hidden을 사용합니다. (O)
  4. clearfix 속성을 만들어서 사용합니다. (O)

컬러를 표현하는 방법

색상, 정확히는 글자의 색상을 의미합니다.

  • 색상명 - red, blue등 이미 정의된 색
  • HEX값 - color:#000, color:#FFFFFF 등의 16진수 색상값
  • RGB값 - rgb(255, 255, 255) 등의 rgb 색상
  • RGBA값 - rgba(200, 100, 150, 0.5) 등의 알파(투명도)가 적용된 rgba 색상값
  • HSL값 - hsl(260%,80%,90%) 각 색상(0~360%), 채도(0~100%), 명도(0~100%)를 의미
  • HSLA값 -hsla(50%,30%,20%,10) 등의 알파(투명도)가 적용된 hsla 색상값
  • Opacity - opacity:0 색상에 대한 알파(투명도)로 0.0~1.0사이의 값

단위를 표현하는 방법

단위는 음의 수를 가질 수 있으며 상대/절대 단위로 구분할 수 있습니다.(숫자 뒤에 공백없이 붙여서 사용하며 0인 경우는 생략할 수 있습니다.)

  • 상대적인 단위
    • em - 요소의 글꼴 크기에 상대적인 값 (2em이란 현재 글꼴 크기의 2배 의미)
    • ex - 현재 글꼴의 X 높이에 상대적인 값
    • ch - "0"의 너비에 상대적인 값
    • rem(root em) - root 요소의 글꼴 크기에 상대적인 값
    • vh - viewport height의 1%에 대한 비율을 나타냅니다
    • vw - viewport width의 1%에 대한 비율을 나타냅니다.
    • vmin - viewport smaller dim ension의 1%에 대한 비율을 나타냅니다.
    • vmax - viewport larger dim ension의 1%에 대한 비율을 나타냅니다.


  • 절대적 단위
    • mm
    • cm - 1cm = 10mm
    • in - 1in = 96px = 2.54cm)
    • px - 컴퓨터 화면에서 사용되는 단위로 1px은 컴퓨터 화면에서 점 1개와 같습니다.
    • pt - 1pt = 1인치의 1/72 (주로 인쇄매체에서 사용)
    • pc - 1pc = 12pt

css를 사용하는 방법

외부선언, 내부선언, 요소 선언 하는 방법이 있습니다.

  • 외부로 선언하는 방법 (External style)
    • <head> 와 </head> 사이에 <link rel="stylesheet" href="css 경로" />

  • 내부로 선언하는 방법 (Internal style)
    • <head> 와 </head> 사이에 <style type="text/css"> </stylel> 넣어서 사용

  • 요소로 선언하는 방법 (Inline style)
    • <div style="width:1000px; margin: 10px 20px 10px 5px;"></div>
    • <td style="width:1000px; margin: 10px 20px 10px 5px;"></td>

절대경로와 상대경로

박스 요소의 바깥쪽 여백을 설정합니다.

  • 절대경로란 어떠한 웹페이지나 파일이 가지고 있는 고유한 경로를 말한다. 즉 절대로 변하지 않는 경로로 절대적 주소를 말한다.
  • 상대경로란 '현재 위치한 곳을 기준'으로 해서 '그곳의 위치'이다.
  • / : 루트 - 가장 최상의 디렉토리로 이동한다.(Web root)
  • ./ : 현재 위치 - 파일이 현재 디렉토리를 의미한다.
  • ../ : 현재 위치의 상단 폴더 - 상위 디렉토리로 이동한다.
  • '../../' : 현재 위치의 상단 상단 폴더 - 두단계 상위 디렉토리로 이동


컨텐츠 요소를 안보이게 하는 방법

컨텐츠 요소를 안보이게 하는 방법

  • display : none; <--> display : block;(영역X)(애니X)
  • opacity : 0; <--> opacity : 1;(영역O)(애니O)
  • visibility : hidden; <--> visibility : visible;(영역O)(애니X)

미디어쿼리(Mediaquery)

미디어쿼리는 화면 크기에 따른 각각의 속성 값을 지정하여, 여러가지 화면을구성하는 기술입니다.

@media [only/not] [media type] [and/,](조건문){실행문}

  • @media : 미디어쿼리 시작을 의미합니다.
  • [only/not] : only는 미디어쿼리를 지원하는 브라우저에서만 해석하고 not은 반대를 의미합니다. 미디어 쿼리 구문을해석하라는 명령어입니다.(생략가능)
  • [media type] : 미디어쿼리를 해석해야 할 대상을 나타냅니다.(생략가능)
    • all : 모든 미디어 유형
    • screen : 컴퓨터 화면
    • print : 인쇄 장치
    • handheld : 휴대용 장비
    • tv : 텔레비젼
    • aural : 음성출력기
    • braille : 점자 출력릭
    • projection : 프로젝트
    • tty : 고정폭을 가지는 텔레타이프
  • [and/,] : and는 둘다 조건에 맞아야 하고, ','는 하나만 맞아도 실행합니다.앞과 뒤에 조건을 나타냅니다.(생략가능)
  • 조건문 : 해당 조건을 설정합니다.
  • 실행문 : 조건에 따른 실행을 설정합니다.

CSS Reference


background

백그라운드 배경 속성 값을 설정합니다.


background-images

백그라운드 이미지를 설정합니다.

  • backround-images: none; - 백그라운드 이미지 속성을 설정하지 않습니다.
  • backround-images: url(이미지 경로); - 백그라운드 이미지 설정합니다.

background-attachment

백그라운드 이미지 고정여부를 설정합니다.

  • backround-attachment: scroll(기본값)
  • backround-attachment: fixed; - 배경이미지를 고정하고 컨텐츠만 움직입니다.
  • backround-attachment: local; - 배경이미지와 컨텐츠가 같이 움직입니다.

background-position

백그라운드 위치를 설정합니다. background-position: x축 y축;

  • background-position: left top;
  • background-position: left center;
  • background-position: left bottom;
  • background-position: right top;
  • background-position: right center;
  • background-position: right bottom;
  • background-position: center top;
  • background-position: center center;
  • background-position: center bottom;
  • background-position: 10px 20px;
  • background-position: 10% 20%;
  • background-position: 10% 20%, 10px 20px, center top;

background-repeat

백그라운드 이미지 반복 여부를 설정합니다.

  • background-repeat: repeat(기본값) - 이미지를 x축 y축으로 반복 설정합니다.
  • background-repeat: repeat-x - 이미지를 x축으로 반복 설정합니다.
  • background-repeat: repeat-y - 이미지를 y축으로 반복 설정합니다.
  • background-repeat: no-repeat - 이미지를 반복하지 않고 하나만 나오게 합니다.

background-size

백그라운드 이미지 사이즈를 설정합니다.

  • background-size: length(단위)
  • background-size: auto(기본값)
  • background-size: contain - 이미지를 요소의 크기에 맞게 설정합니다. 가로와 세로 비율을 유지하기 때문에 공백이 생길 수 있습니다.
  • background-size: cover - 이미지를 요소의 크기에 맞게 설정합니다. 공백 없이 화면 크기에 맞추기 때문에 이미지가 왜곡 될 수 있습니다.(contain 보다 cover을 많이 씀)

background-blend-mode

백그라운드 이미지 브랜드 모드를 설정합니다.


background-color

백그라운드의 색을 정합니다.

  • background-color: transparent; =백그라운드를 투명하게 설정합니다.(기본값)
  • background-color: 컬러; =백그라운드의 컬러값을 설정합니다.

color

텍스트의 색을 정합니다.

  • color: 컬러; =# + 6자리 컬러코드 (반복되는 컬러코드는 3자리로 표현 가능)

border

테이블 테두리 입니다.

  • border: 2px solid #333;

border-color

테두리의 색상으로, 값은 color 속성의 포맷을 사용합니다.

  • border-color: #333;
  • border-color: red;

border-width

테두리의 두께로, 주로 px 단위를 사용합니다.

  • border-width: 2px; - 상, 하, 좌, 우 모두 2px
  • border-width: 2px 3px; - 상,하2px 좌,우3px
  • border-width: 2px 3px 4px; - 상2px 좌,우3px 하4px
  • border-width: 2px 3px 4px 5px; - 상2px 우3px 하4px 좌5px
  • border-width: thin; - 단선(=1px)으로 표시 합니다.
  • border-width: medium; - 보통 두께(중간 굵기 - 기본값)로 표시 합니다.
  • border-width: thick; - 굵은 선으로 표시 합니다.


border-style

테두리의 스타일로 실선, 점선, 이중선 등의 옵션이 존재합니다.

  • none - 초기값입니다. 보더는 표시되지 않고, 선폭도 "0"이 됩니다. 테이블 셀 등의 보더가 겹치는 경우 다른 값이 우선됩니다.
  • solid - 단선(실선)으로 표시됩니다.
  • double - 복선으로 표시됩니다.
  • hidden - 보더는 표시되지 않고 두께도 "0"이 됩니다. 테이블 셀 등의 보더가 겹치는 경우, hidden이 우선시 됩니다.
  • groove - 입체적으로 움푹 패어 들어가 보이는 선으로 표시합니다.
  • ridge - 입체적으로 선이 돌출되어 표시됩니다.
  • inset - 보더(border)로 둘러싸인 영역 전체가 입체적으로 움푹 들어간 것 같이 표시됩니다.
  • outset - 보더(border) 둘러싸인 영역 전체가 입체적으로 돌기되어 보이는 것 같이 표시됩니다.
  • dotted - 점선으로 표시됩니다.
  • dashed - 파선(굵은 점선)으로 표시됩니다.

  • border-image-repeat: space; - 원본 이미지의 모서리 영역을 타일처럼 반복해 간격을 채웁니다. 크기가 맞지 않으면 각 이미지에 균등하게 공백을 배치합니다.

  • border-image-slice: 10% 30%; - 첫 번째 값은 위와 아래 분할선, 두 번째 값은 왼쪽과 오른쪽 분할선의 위치를 설정합니다.
  • border-image-slice: 30px 30% 45px; - 첫 번째 값은 위, 두 번째 값은 왼쪽과 오른쪽, 세 번째 값은 아래 분할선의 위치를 설정합니다.
  • border-image-slice: 7px 12px 14px 5px; - 각각 상, 우, 하, 좌 분할선의 거리를 순서대로 지정합니다.(시계방향)
  • border-image-slice: 10% fill 7px 12px;
    중앙 영역을 보존하여 배경 이미지처럼 사용. 실제 background 속성보다 위에 그려집니다.
    중앙 영역 이미지는 위와 왼쪽 모서리 영역의 크기에 맞도록 늘어납니다.

  • w-resize - 좌우 화살표표 모양의 마우스 커서
  • wait - 동작중임을 나타내는 모래시계 마우스 커서
  • zoom-in - 돋보기 + 모양의 마우스 커서
  • zoom-out - 돋보기 - 모양의 마우스 커서
  • initial - 초기화 커서
  • inherit - 커서 속성의 값이 부모 요소에서 상속하도록 지정
커서 모양보기

display

어떻게 보여줄지를 지정하는 속성입니다.

  • none - 요소를 아예 보여주지 않습니다.(공간차지X)
  • inline - 요소를 인라인 요소로 보여줍니다.
  • block - 요소를 블록요소로 보여줍니다.
  • inline-block - 요소를 인라인-블록(두 가지 특징을 동시에 가진 형태)로 보여줍니다. 마진, 테두리, 패딩, 너비, 높이 속성이 적용됩니다.
  • flex - CSS3에 새로 생긴 값으로 블록-레벨의 flex 컨테이너처럼 요소를 표현합니다.
  • inline-flex - CSS3에 새로 생긴 값으로 인라인-레벨의 flex 컨테이너처럼 요소를 표현합니다.
  • list-item - 박스를 블록 처리하고 그 안의 아이템들을 인라인 박스 형태로 처리합니다.
  • run-in - 문맥에 따라 블록 또는 인라인 박스로 처리됩니다.
  • table, inline-table, table-caption, table-cell, table-column, table-column-group, table-header-group, table-footer-group, table-row, table-row-group - 이 속성값들은 테이블 스타일의 요소의 동작을 정의합니다.

visibility

문서의 레이아웃을 변경하지 않고 요소를 보이거나 숨깁니다.visibility로 <table>의 행이나 열을 숨길 수도 있습니다.

  • visible - 기본 값으로, 요소를 보여줍니다.
  • hidden - 요소를 보여주지 않습니다.(공간차지O)
  • collapse - table 요소들에서만 줄 수 있는 값으로, hidden과 비슷하지만, 필요에 따라 공백을 없애서 테이블 레이아웃을 자연스럽게 유지합니다.
  • inherit(기본값) - 부모 요소의 값을 상속 합니다.

overflow

요소내의 컨텐츠가 너무 커서 요소내에 모두 보여주기 힘들때 그것을 어떻게 보여줄지를 지정합니다.

  • visible - 넘칠 경우 컨텐츠가 상자 밖으로 보여집니다.(기본값)
  • hidden - 넘치는 부분은 잘려서 보여지지 않습니다.
  • scroll - 스크롤바가 추가되어 스크롤할 수 있습니다.(가로, 세로 모두 추가 됩니다.)
  • auto - 컨텐츠 량에 따라 스크롤바를 추가할지 자동으로 결정됩니다.( 필요에 따라 가로, 세로 별도로 추가될 수도 있습니다.)
  • overflow-x, overflow-y - 콘텐츠가 수평 혹은 수직 방향으로 넘칠 때의 처리 방식을 결정합니다.

text-overflow

박스 안에 내용이 넘칠 때 텍스트를 어떻게 처리할지 지정하는 속성입니다.

  • clip - 텍스트를 자릅니다.(기본값)
  • ellipsis - 잘린 텍스트를 생략 부호(...)로 표시합니다.
  • string - 잘린 텍스트를 지정한 문자열로 표시합니다. 예)div {text-overflow:"---";} *string의 경우, 현재 지원하는 브라우저가 없음*

white-space

스페이스와 탭, 줄바꿈, 자동줄바꿈을 어떻게 처리할지 정하는 속성입니다.

  • normal - 연속된 공백과 줄 바꿈은 메꾸어져서 하나의 공백으로 표시됩니다.
  • nowrap - normal과 같이 공백을 채우지만 가로로 긴 줄에서도 줄 바꿈을 하지 않고 표시합니다.
  • pre - 연속된 공백을 그대로 표시합니다. 또한 긴 줄도 줄 바꿈하지 않고 표시합니다.
  • pre-wrap - 연속 공백은 그대로 유지됩니다. 행(行)의 줄 바꿈은 행의 박스를 채우기 위해 필요한 경우에 실행합니다.
  • pre-line - 연속 공백은 메꾸어져 하나의 공백이 됩니다. 행의 줄 바꿈은 행의 박스를 메꾸기 위해 필요한 때에 실행됩니다.

해상도에 따른 이미지 처리하는 방법

<img>태그를 이용하는 방법
- srcset과 sizes 속성을 추가하여 반응형 이미지 처리
- srcset : srcset을 지원하는 브라우저는 이 값을 읽어 현재의 화면 상태에 맞는 적절한 이미지를 불러옵니다.
- sizes: 미디어쿼리의 중단점(breakpoint)별로 필요한 이미지의 정보를 추가로 제공함으로써 브라우저로 하여금 현재의 상태에 맞는 이미지를 불러옵니다.

srcset="assets/img/blog_img01_1x.jpg 1x,
assets/img/blog_img01_2x.jpg 2x,
assets/img/blog_img01_3x.jpg 3x,
assets/img/blog_img01_4x.jpg 4x" alt="블로그 이미지">
